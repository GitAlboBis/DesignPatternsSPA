# 🎯 Design Patterns Study SPA

Una Single Page Application (SPA) interattiva sviluppata interamente in HTML, CSS e Vanilla JavaScript, pensata come strumento di supporto per lo studio e il ripasso dell'esame di Ingegneria del Software.

## 🚀 Funzionalità Principali

* **Libreria Completa:** Tutti i 23 Design Pattern della Gang of Four (Creazionali, Strutturali, Comportamentali).
* **Confronto Visivo:** Diagrammi UML strutturali affiancati a diagrammi UML di esempi applicativi concreti per ogni pattern.
* **Analisi Rapida:** Sezioni dedicate a definizioni, vantaggi (Pro) e svantaggi (Contro).
* **Confronti Teorici:** Un sistema ad *accordion* per esplorare le sottili differenze teoriche tra pattern simili (es. State vs Strategy, Decorator vs Proxy).
* **Zero Dipendenze:** Nessun framework complesso o processo di build richiesto. Basta aprire `index.html` nel browser.

## 🛠️ Tecnologie Utilizzate
* HTML5
* CSS3 (Flexbox/Grid per layout responsivo)
* Vanilla JavaScript (ES6+)
